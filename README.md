## How to Run

### C – Bubble Sort
```bash
gcc C/bubble_sort.c -o bubble_sort
./bubble_sort
```

### C++ – Binary Search
```bash
g++ CPP/binary_search.cpp -o binary_search
./binary_search
```

### Java – Merge Sort
```bash
javac Java/MergeSort.java
java -cp Java MergeSort
```

### Python – Quick Sort
```bash
python3 Python/quick_sort.py
```
